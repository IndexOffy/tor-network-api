### Running the project with [Docker]

 - Building the Docker image

```bash
$ sudo docker build --tag tor-api --file docker/Dockerfile .
```

 - Starting the Docker Container

```bash
$ sudo docker run -d -t -p 8080:8080 tor-api/dev
```
